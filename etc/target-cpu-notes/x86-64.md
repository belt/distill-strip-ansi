The `x86-64` baseline (v1, 2003+) guarantees only SSE2 — the
minimum for any 64-bit x86 chip since AMD K8 (2003). No
AVX, no FMA, no POPCNT. Compiler auto-vectorization is
limited to 128-bit SSE2 lanes. `memchr` still uses runtime
SIMD detection (AVX2 if available), so the hot-path scanner
is unaffected — but palette matrix math and downgrade
distance calculations won't benefit from FMA or wider
registers. Consider `x86-64-v3` if all target hardware is
Haswell (2013) or newer.
