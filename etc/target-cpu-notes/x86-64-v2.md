The `x86-64-v2` level (2008+, Nehalem) adds SSE4.2, POPCNT,
and CMPXCHG16B over the v1 baseline. SSE4.2 enables
hardware `pcmpestri`/`pcmpistrm` for string operations,
and POPCNT is used by some hash and bitfield algorithms.
Still no AVX or FMA — auto-vectorization stays at 128-bit
width. `memchr` runtime-detects wider SIMD independently.
Suitable for VMs and cloud instances that expose only the v2
feature set (common on AWS Graviton-less x86 fleets pre-2020).
