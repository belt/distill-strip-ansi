The `x86-64-v4` level (2017+, Skylake-X) requires AVX-512F,
AVX-512BW, AVX-512CD, AVX-512DQ, and AVX-512VL — Ice Lake,
Ice Lake, Zen 4+. Enables 512-bit auto-vectorization for
loops the compiler can prove safe. Caution: AVX-512 can
cause frequency throttling on some Intel SKUs (different
license levels per core type on Alder Lake+). Benchmark
with and without to confirm net throughput gain. `memchr`
already uses AVX2 at runtime; the v4 flag primarily helps
palette matrix math and bulk byte transforms.
